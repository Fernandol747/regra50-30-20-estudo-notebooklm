# regra50-30-20-estudo-notebooklm
Projeto da DIO com notebookml sobre a regra 50/30/20 da gestão de dinheiro

# Contexto e Objetivos:
O assunto escolhido para o caderno temático foi a estratégia 50/30/20 sobre o dinheiro, onde 50% são para necessidades, 30% para desejos e 20% para investimentos. O objetivo deste projeto consiste no desenvolvimento de uma aplicação (disponível em plataforma web/mobile) voltada para a gestão e organização financeira pessoal. A solução fundamenta-se na metodologia de divisão orçamentária 50/30/20, visando proporcionar ao usuário uma visão clara e automatizada da alocação de sua renda entre necessidades básicas, gastos não essenciais e reservas de investimento. A ferramenta busca otimizar a tomada de decisão financeira e promover o consumo consciente por meio de uma interface intuitiva e recursos de acompanhamento em tempo real.
